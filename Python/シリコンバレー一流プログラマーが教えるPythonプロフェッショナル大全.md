# Lesson1 Pythonの基本
- 変数の宣言とか数値計算のやり方について  
「変数とはなんぞや」とか「型とはなんぞや」みたいな話はせず  
「Pythonでは変数はこう宣言します」で入るので全く初めてプログラムやる人向けって感じではない。

- 文字列操作について  
  「こんな関数あるで」っていろんな操作を紹介していく感じっぽい。  
  「`count`を使うと文字列の数を返してくれます」みたいなのが沢山あるのでレファレンス的に使えるかも。
